# 我想变成凑企鹅 (TomoriPenguin)

🐧 **《我想变成凑企鹅》** 是一款基于 Web 的硬核心理学模拟视觉小说（Galgame）。在游戏中，玩家将扮演羽丘女子学园新来的心理辅导干事，通过对话与心理干预，帮助缺乏安全感、极度社恐的高松灯（《BanG Dream! It's MyGO!!!!!》）逐步克服内心的恐惧，最终觉醒成为自信的“凑企鹅”。

## ✨ 游戏特色

*   **🧠 硬核心理学干预系统**：游戏中的每一个选项都对应着真实的心理学干预技巧（如情绪确认、无条件积极关注、赋能等）。
*   **⚠️ 真实的失败复盘**：如果选择了错误的选项导致 Bad End，游戏会弹出详细的“心理干预失败报告”，从专业角度分析你的错误行为、对灯造成的负面影响以及正确的心理学理论（涉及情绪无效化、习得性无助等概念）。
*   **📈 企鹅度 (Penguin Degree)**：引入了独特的进度条系统，实时反映高松灯内心的成长与自信程度。数值达到 100% 即可解锁真结局。
*   **🎭 丰富的多结局与多媒体演出**：包含多条分支剧情、全程角色语音配合、立绘差分以及通关后的隐藏随机 CG 视频演出。
*   **📂 灵活的素材管理**：采用结构化的资产目录，方便快速加载和自定义替换背景、立绘、语音和视频素材。

## 🛠️ 技术栈

*   **前端**：HTML5, CSS3, 原生 JavaScript (Vanilla JS)
*   **架构**：自定义轻量级 Galgame 引擎 (`engine.js` 引擎核心 + `script.js` 剧本数据分离)

## 🚀 游玩方法

本项目纯前端实现，无需复杂的环境配置：

1.  克隆或下载本仓库到本地。
2.  使用现代浏览器（推荐 Chrome、Edge 或 Firefox）直接打开 `index.html` 文件即可开始游戏。
    *   *注：由于浏览器的本地跨域安全策略（CORS），如果在本地直接双击打开遇到音视频加载问题，建议使用 VS Code 的 Live Server 插件或运行任意本地 HTTP 服务器。*

## 📁 项目结构

```text
TomoriPenguin/
├── index.html       # 游戏主入口文件
├── css/
│   └── style.css    # 游戏样式表
├── js/
│   ├── engine.js    # 游戏核心引擎逻辑
│   └── script.js    # 游戏剧本与节点数据
└── assets/          # 游戏资源目录 (支持自定义替换)
    ├── bg/          # 背景图片
    ├── sprites/     # 角色立绘
    ├── voices/      # 角色语音
    └── videos/      # 结局 CG 视频
```

## 📝 剧本说明

游戏剧本存放于 `js/script.js` 中，采用 JSON 树状节点结构编写。你可以通过修改该文件轻松扩展剧情、添加新的选项分支或调整企鹅度数值。
